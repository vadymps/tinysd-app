# TinySD API

Backend API for the TinySD application.

## Supported Providers

- `modelslab`
- `stability_ai`
- `google_gemini`
- `huggingface_inference`

## Setup

```bash
npm install
```

## Development

```bash
# development
npm run start:dev

# production build
npm run build

# production run
npm run start:prod
```

## Tests

```bash
npm test
npm run test:e2e
npm run test:cov
```

## Lint

```bash
npm run lint
```
