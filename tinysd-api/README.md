# TinySD API

Backend API for the TinySD application.

## Setup

```bash
npm install
```

## Development

```bash
# development
npm run start:dev

# production build
npm run build

# production run
npm run start:prod
```

## Tests

```bash
npm test
npm run test:e2e
npm run test:cov
```

## Lint

```bash
npm run lint
```
